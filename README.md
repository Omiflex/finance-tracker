# Student Finance Tracker 📚

A responsive web application built for our Web Programming course to help university students manage their personal finances, budget efficiently, and save for their goals.

## 🌟 Key Features
* **User Authentication**: Secure registration and login portals for students
* **Expense Tracking**: Easily log daily spending with descriptions, amounts, dates, and categories
* **Category Budgeting**: Set monthly allowances for categories like Food, Transport, and Education while tracking remaining balances
* **Savings Goals**: Set financial milestones, contribute money toward them, and track target deadlines

## 🛠️ Project Structure & Tech Stack
This project uses a clean client-server model split across the files:
* **Frontend**: Interactive responsive UI components built using HTML5, CSS3, and JavaScript
* **Backend**: A central PHP API handling database operations and routing requests via JSON communication
* **Database**: MySQL relational schema processing tables for Users, Expenses, Budgets, and Savings Goals
