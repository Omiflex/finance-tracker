# Student Finance Tracker 📚💰

A responsive web application built for our Web Programming course to help university students manage their personal finances, budget efficiently, and save for their goals.

## 🌟 Key Features
* **User Authentication**: Secure registration and login portals for students[cite: 1, 4, 5].
* **Expense Tracking**: Easily log daily spending with descriptions, amounts, dates, and categories[cite: 1, 2].
* **Category Budgeting**: Set monthly allowances for categories like Food, Transport, and Education while tracking remaining balances[cite: 1, 2].
* **Savings Goals**: Set financial milestones, contribute money toward them, and track target deadlines[cite: 1, 2].
